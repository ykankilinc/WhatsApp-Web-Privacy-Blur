# WhatsApp Web Privacy Blur Extension

A privacy-focused Chrome extension for WhatsApp Web that allows you to blur messages, media, and contact details. Features a robust Screen Lock for enhanced security.

## Features

### üîí Privacy Blur
Granular control over what you want to hide:
- **Blur All Messages**: Blurs chat bubbles and message text.
- **Blur Last Messages**: Blurs the message preview in the sidebar.
- **Media Preview**: Blurs images, videos, stickers, and GIFs in the chat.
- **Media Gallery**: Blurs media in the drawer/info panel.
- **Text Input**: Blurs the typing area until you hover.
- **Profile Pictures**: Blurs all contact photos and avatars.
- **Group/User Names**: Blurs names in headers and the contact list.
- **No Hover Delay**: Option to unblur items instantly on hover without a transition.

Protect your WhatsApp Web session from prying eyes:
- **Password Protection**: Set a custom password to lock the interface.
- **Auto-Lock**: Automatically locks the screen after a set period of inactivity (default: 5 mins).
- **Manual Lock**: "Lock Now" button for instant security.
- **Shortcut Lock**: Press `Alt + Shift + L` to instantly lock the screen.
- **Popup Security**: The extension settings menu is also password-protected when Screen Lock is enabled.

## Installation

### From Source (Developer Mode)
1.  Download or clone this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (top right toggle).
4.  Click **Load unpacked**.
5.  Select the folder containing this extension.

## Usage
1.  Click the extension icon in the Chrome toolbar.
2.  Use the master toggle to enable/disable the extension.
3.  Customize your privacy preferences using the individual switches.
4.  **For Security**: 
    - Go to the "Screen Lock Security" section.
    - Check "Enable Screen Lock".
    - Set a password.
    - Adjust the "Auto-Lock" timer if needed.
